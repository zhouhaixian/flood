# flood

短信之洪水

## 环境依赖

- ImageMagick
- Tesseract-OCR

在 Linux ( Ubuntu ) 下安装:

```bash
sudo apt-get install imagemagick
sudo apt-get install tesseract-ocr
```

在 Windows 下安装:

**ImageMagick**: https://imagemagick.org/script/download.php#:~:text=an%20iOS%20application.-,Windows%20Binary%20Release,-ImageMagick%20runs%20on

**Tesseract-OCR**: https://github.com/UB-Mannheim/tesseract/wiki#tesseract-installer-for-windows
