# flood

短信之洪水

## 开发

```bash
# 安装依赖
yarn install
```

### 调试

如果你使用 `VSCode`, 只需打开需要调试的 `API` 文件, 并按下 `F5` 键, 即可开始调试

或者, 你也可以运行 `yarn debug <API名称>`, 即可开始调试, 例如: `yarn debug user.mofcom.gov.cn`

### 运行

```bash
yarn start
```

## 编译

```bash
yarn build
```

编译成功后，可执行文件位于 `dist/bin` 目录下
